# FuzzTester
### Introduction
**FuzzTester** is a pentesting tool can send a randomize payloads to trigger the vulnerability
### Installation
```
$ git clone https://github.com/fafsnords/FuzzTester
$ cd FuzzTester

$ pip install -r requirements

$ python fuzz.py
```
