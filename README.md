# FuzzTester
### Introduction
**FuzzTester** is a pentesting tool that used for send a randomize payloads to trigger the vulnerability
### Installation
```
$ git clone https://github.com/fafsnords/FuzzTester

$ cd FuzzTester

$ pip install -r requirements

$ python fuzz.py
```
